	tools i will need to develop are below:
depend: a tool that gathers markdown, etc and connects my go tools together (meta but will be how dependencies folder is created) 
nav: a header that allows you to navigate and view folders and markdown files similar to sourcehut (will include ~/folders navigation, timestamps, permalinks, organization

footy: like the header, footer  is simple and may contain relevant links (generated depending on architecture)
oigit: tiny git protocol that helps me develop and mirror structure of site using git. version control (can include link to previous versions through site)
